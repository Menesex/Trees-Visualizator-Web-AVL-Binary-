# 🌳 TreeVisualizer: BST vs AVL ⚖️

Hi! I'm **Juan José Meneses Jaramillo (MeneDev)**. This is an educational project designed to visualize and understand the fundamental differences between a standard **Binary Search Tree (BST)** and a **Self-Balancing AVL Tree**.

## 🔗 Try it live!
You can interact with the visualizer without installing anything here:
👉 **[https://menetreesvisualizator.streamlit.app]**

---

## 🚀 About the Project
In software engineering, efficiency is key. This project allows you to see in real time how an **AVL** tree uses automatic rotations to stay optimized, comparing it with the rigid structure of a conventional **BST**.

### ✨ Features
* **Dynamic Visualization:** Graphs generated with `Graphviz`.
* **Full Interactivity:** Insert values (positive or negative) and watch the nodes rearrange.
* **Educational Analysis:** Displays heights and traversal orders.
* **Modern Interface:** Built with `Streamlit`.

## 🛠️ Technologies
* **Python 3.14** 🐍
* **Streamlit** (Web Framework)
* **Graphviz** (Graph Engine)

## 📥 Local Usage (Quick)
If you prefer to run it on your PC:

1. **Install dependencies:** `pip install streamlit graphviz`
2. **Run the AVL:** `streamlit run AVL_Tree.py`
3. **Run the BST:** `streamlit run Binary_Tree.py`
